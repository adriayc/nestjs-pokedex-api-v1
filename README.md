# POKEDEX API (NestJS, MongoDB and Docker)

## NestJS

[NestJS](https://nestjs.com/) es un framework que permite desarrollar servidores y APIs usando una estructura modular, con inyección de dependencias y buenas prácticas listas desde el inicio.

## Docker

[Docker](https://www.docker.com/) es una plataforma que permite empaquetar una aplicación con todo lo que necesita para funcionar (código, dependencias, configuración) en un contenedor.

### Docker Desktop

[Docker Desktop](https://www.docker.com/products/docker-desktop/) es una aplicación con interfaz gráfica que te permite usar Docker fácilmente en tu computadora.

## MongoDB

[MongoDB](https://www.mongodb.com/) es una base de datos NoSQL que almacena datos en formato de documentos (tipo JSON).

### MongoDB Compass

[MongoDB Compass](https://www.mongodb.com/products/tools/compass) es la herramienta oficial con interfaz gráfica para explorar, visualizar y administrar bases de datos MongoDB.

### TablePlus

[TablePlus](https://tableplus.com/) es una aplicación gráfica que te permite conectarte, visualizar y gestionar bases de datos de forma sencilla.
